## 🚀 Featured Projects

### 📱 FideliCards | End-to-End

All-in-one digital wallet for loyalty cards, designed and developed end-to-end.

- 🔐 Secure authentication with **Google & Apple Sign-In**
- ☁️ Cloud-native architecture on **Azure** (Container Apps, ACR, Azure SQL)
- 🐳 Backend API built with **.NET**, containerized with Docker
- 📱 Cross-platform mobile app built with **React Native** (iOS live on App Store, Android in testing)
- 🏗️ Focus on scalability, security, and performance

👉 Available on the App Store -> [FideliCards on AppStore](https://apps.apple.com/app/fidelicards/id6759812207)
